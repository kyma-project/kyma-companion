# Issues with APIRules and Service Connection

If you have issues creating an APIRule custom resource (CR) or you've exposed a service but you cannot connect to it, see the troubleshooting guides related to:

- [APIRule CR in version v2](./v2/03-00-basic-diagnostics.md)
- [v1beta1 management issues](./03-70-apirule-v1beta1-management-issues.md)

> [!WARNING]
> APIRule CRD v2 is the latest stable version. Version v1beta1 is removed in release 3.4 of the API Gateway module.
> You can no longer create new v1beta1 APIRules. To make any changes, you must migrate to version v2.
> For the migration procedure, see [APIRule Migration](./apirule-migration/).
