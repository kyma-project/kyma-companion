# Issues with APIRules and Service Connection

If you have issues creating an APIRule custom resource (CR) or you've exposed a service but you cannot connect to it, see the troubleshooting guides related to:

- [APIRule CR in version v2](./v2/03-00-basic-diagnostics.md)

> [!WARNING]
> APIRule in version `v1beta1` is removed in release 3.4 of the API Gateway module. Existing v1beta1 APIRules cannot be edited or deleted and must be migrated to version `v2`. For migration guidance, see the [APIRule migration documentation](https://kyma-project.io/#/api-gateway/user/custom-resources/apirule/04-10-apirule-custom-resource).
