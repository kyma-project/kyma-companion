# APIRule Custom Resource <!-- {docsify-ignore-all} -->

The `apirules.gateway.kyma-project.io` CRD describes the kind and the format of data the APIRule Controller uses to configure resources. The stable and recommended APIRule version is `v2`.

> [!WARNING]
> APIRule in version `v1beta1` is removed in release 3.4 of the API Gateway module. You can no longer create new v1beta1 APIRules, and existing ones cannot be deleted or edited — they must be migrated to version `v2`. For more information, see the [APIRule Migration](https://kyma-project.io/#/api-gateway/user/custom-resources/apirule/04-10-apirule-custom-resource).

Browse the documentation related to the APIRule CR in version `v2`:
- [Specification of APIRule CR](./v2/04-10-apirule-custom-resource.md)
- [APIRule Access Strategies](./v2/04-15-api-rule-access-strategies.md)
